1. 前（fei）言（hua）

此篇基础教程针对纯小白Mac用户。由于Mac上某些软件的缺失，或者linux开发环境的需要，Mac上很多时候还是需要安装一个虚拟机的。Mac比较常见虚拟机平台有Parallel Desktop和VMware Workstation，我习惯用前者，所以教程还是以PD为例。更多细节可能还是在安装Windows7部分讲解，所以想直接看怎么安装Ubuntu的同学最好从头到尾都浏览一遍。

2. 安装Parallel Desktop

Parallel Desktop是MacOS平台大名鼎鼎的虚拟机软件平台，第一次被他惊艳到是因为他的融合模式，不过后来发现其他的虚拟机软件也都有这个功能了。话不多说，先来看看怎么安装。
首先，我们需要下载Parallel Desktop TNT版，为什么用TNT版，就不多解释了，毕竟官方的很贵嘛，咳咳；
点击下载 提取密码:qt4Dy

  • 安装前请断开Mac的网络
  • 首先双击dmg加载安装包镜像，双击安装，没有网络会提示无法检测新版本，果断点击安装当前的
  • 参与体验计划之类的提示，习惯性点否，安装完成后，会自动进入软件

3. 安装Windows7

  • 现在来创建虚拟机系统，首先选择从镜像文件新建
  • 点击手动查找，然后点击选择文件
  • 这里选择已经下载好的wi7镜像，必须是ISO格式的，别下载什么GHOST的，原版镜像下载：MSDN I tell you
  • 继续，这里按图中勾选，新手没什么特别需要直接选择快速安装就好，密钥也不用填，回头进了系统再想办法激活，当然不激活也一样用
  • 继续，这里选择虚拟机系统用途，不同的用途会分配不同的配置，正常的选择软件开发或者生产力测试就好
  • 继续，这里也按图中勾选
  • 继续，这时候稍微等一会就能自动进入安装界面，如果选择的windows安装镜像包含多个版本的windows，就会让你选择需要安装的操作系统，一般为了节省资源，选择家庭普通版就好
  • 点击下一步，然后等待一段时间，系统会自动安装，中间会重启几次
  • 完了就直接进入系统了，看到熟悉的崭新的windows7桌面，后面正常使用就好

4. 安装Ubuntu

  • 安装Ubuntu就更简单了，首先和安装windows7不同的是，需要选择不同的镜像文件
  • 自己选择输入用户名和密码，这个挺重要的，一定要记住
  • 然后就需要很长时间的等待，Ubuntu全程自动安装
  • 安装完成之后就可以顺利看到Ubuntu登陆界面啦，输入刚才设置的密码进入桌面，然后开始正常使用
